¿Dónde vas a instalarlo?

FacturaScripts es básicamente una aplicación web, como un blog, y para instalarlo necesitas un servidor web. Puedes instalarlo en tu propio ordenador, o bien puedes instalarlo en la nube (un hosting) y tenerlo disponible siempre online desde cualquier parte del mundo.
Tú decides ;-)

Descargar
  1. Descarga el ejecutable.
  2. Instala dónde quieras.
  3. Ejecuta configurar.bat
  4. Ejecuta iniciar.exe
  5. Entra en http://localhost

Licencia y código

El código fuente de FacturaScripts está disponible en Github, tiene licencia Lesser GPL 3, es el mismo código que descargas y puedes obtenerlo aquí: https://github.com/NeoRazorX/facturascripts_2015

Eso si, cada plugin tiene su propio repositorio, normalmente en GitHub, pero no siempre. Puedes ver el repositorio de cada plugin desde la web de ese plugin.

Si quieres modificar FacturaScripts o alguno de sus plugins, no olvides que tienes que copiar los archivos que modifiques a tu propio plugin, sino cuando haya actualizaciones perderás los cambios.
